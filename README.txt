Restaurant menu web site.

Free OpenSource programm.

Web site represented restaurant menu where admin can create, read, update and delete differen dishes. 

User can see menu options with name of cousin, cost and short description.

Files:
		1) database_setup.py contain configuration, table and mapper represented in python classes. Created DB, used SQL Alchemy ORM.
        2) restaurantmenu.db sqlite database file 

To use this system you should have:

		- Python 3.4 or higher
		- sqlite, postgre SQL 9.3.13 version or other database
		- SQL Alchemy python ORM
        - Flask web micro framework for Python
